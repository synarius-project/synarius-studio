Studio Components Overview
==========================

This section contains requirements for synarius-studio subcomponents only.

Studio component codes:

- ``GUI`` for gui
- ``EDITOR`` for editor
- ``VIS`` for visualization

Component requirement IDs must use:

- ``STUDIO-COMP-GUI-NNN``
- ``STUDIO-COMP-EDITOR-NNN``
- ``STUDIO-COMP-VIS-NNN``
